HC-SR04 Ultrasonic Sensor

Overview

The HC-SR04 ultrasonic sensor is used for obstacle detection in the robot car.

Purpose in the Project

- Detects nearby obstacles.
- Measures distance between robot and objects.

Technical Specifications

Parameter| Details
Operating Voltage| 5V
Measuring Range| 2cm – 400cm
Accuracy| ±3mm
Frequency| 40kHz

Pin Connections

Sensor Pin| NodeMCU Pin
VCC| VIN
GND| GND
TRIG| D5
ECHO| D6

Working Principle

The sensor emits ultrasonic waves and measures the time taken for the echo to return. Distance is calculated based on travel time.

Advantages

- Accurate distance measurement.
- Low cost.
- Easy integration with microcontrollers.
